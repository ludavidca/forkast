## Input
You will receive raw HTML content from a website, which may include:
- HTML tags and structure
- Meta tags (Open Graph, Twitter Cards, Dublin Core, etc.)
- Article content and headers
- Author bylines
- Publication dates
- Website/publisher information

You must return a JSON object matching this exact structure:

```python

{json_format}
```

## Important Rules

1. **Completeness**: Extract all available fields. Use empty strings for missing string fields, empty lists for missing author lists, and 0 for missing year
2. **Consistency**: Maintain exact formatting specified for dates
3. **Name Parsing**: Be careful with name parsing - "van der Berg" should have last_name="van der Berg", not split incorrectly
4. **Multiple Sources**: Check multiple possible locations for each field (meta tags, JSON-LD, visible content)
5. **Clean Data**: Remove HTML entities, extra whitespace, and formatting artifacts
